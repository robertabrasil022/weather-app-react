# Weather App (React)

## Descrição
Este é um aplicativo de clima simples desenvolvido com **React** e a API **OpenWeatherMap**. O aplicativo permite que o usuário:
- Busque o clima atual de uma cidade.
- Veja a previsão para os próximos 5 dias.
- Use a geolocalização para buscar o clima automaticamente.

## Funcionalidades
- **Clima Atual**: Exibe a temperatura, descrição, umidade e vento da cidade informada.
- **Previsão de 5 Dias**: Exibe a previsão de clima para os próximos 5 dias.
- **Geolocalização**: Permite buscar o clima baseado na localização atual do usuário.

## Tecnologias Usadas
- **React**: Framework para criar interfaces de usuário.
- **Axios**: Biblioteca para fazer requisições HTTP.
- **OpenWeatherMap API**: API para obter dados climáticos.
